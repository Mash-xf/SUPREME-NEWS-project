# Supreme New

## 🌍 Project Overview
**Supreme New** is a web-based application developed by **Felix Macharia** that displays real-time news from all over the world. The platform helps users stay updated with the latest events, trends, and activities happening globally.

---

## 📁 Project Structure
```
supreme-new/
│
├── index.html      # Main homepage displaying news
├── about.html      # Information about the project
├── contact.html    # Contact page
├── style.css       # Styling for the website
├── news.js         # JavaScript logic for fetching and displaying news
└── README.md       # Project documentation
```

---

## 🚀 Features
- 📰 Real-time news updates  
- 🌐 Global news coverage  
- ⚡ Simple and responsive design  
- 📱 Accessible on any modern browser

## 📦 Installation
This project **does not require installation**.

You can access and run it easily using any web browser.

---

## ▶️ Running the Project
1. Download or clone the repository:
   ```bash
   git clone https://github.com/your-username/supreme-new.git
   ```

2. Open the project folder.

3. Double-click on `index.html`  
   **OR** right-click and open it with your browser (Chrome, Firefox, Edge, etc.).

The website will run instantly without any setup.
